# SonDepremler
Türkiye'deki Depremleri canlı izleyebileceğiniz Python Uygulaması.

## Kurulum
1. `git clone https://github.com/bavertorun/SonDepremler.git`

2. `cd SonDepremler`

3. `python SonDepremlerV1`

4. `pip install -r requirements.txt`

5. `python main.py`
